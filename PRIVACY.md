# 개인정보 처리방침

시행일: 2026-05-13

JobFilter for Korea Jobs는 한국 채용 사이트의 공고를 필터링하고 지원 이력을 관리하기 위한 Chrome 확장 프로그램입니다.

## 수집 및 저장하는 데이터

JobFilter는 사용자가 확장 프로그램을 사용할 때 다음 정보를 브라우저에 저장할 수 있습니다.

- 제외 업체명
- 회사명 비교를 위한 정규화된 회사명
- 지원 현황 페이지에서 수집한 지원 이력
- 회사명, 공고명, 지원일, 지원 상태, 열람 여부, 지원 플랫폼, 공고 URL
- 제외 업체 숨김 및 지원완료 업체 숨김 같은 확장 설정

이 데이터는 사용자가 직접 우클릭 메뉴, 수집 버튼, 직접 추가, CSV 등록 등을 사용할 때 생성되거나 저장됩니다.

## 저장 위치

모든 데이터는 사용자의 브라우저 안에 있는 `chrome.storage.local`에 저장됩니다.

JobFilter는 별도의 외부 서버를 운영하지 않으며, 저장된 지원 이력이나 제외 업체 목록을 외부 데이터베이스로 전송하지 않습니다.

## 제3자 제공

JobFilter는 사용자의 데이터를 판매, 대여, 공유하지 않습니다.

지원 이력, 제외 업체 목록, 설정 정보는 개발자에게 전송되지 않습니다.

## 사이트 접근 권한 사용

JobFilter는 지원하는 채용 사이트에서 다음 기능을 제공하기 위해 사이트 접근 권한을 사용합니다.

- 공고 목록에서 회사명과 공고 정보를 읽기
- 제외 업체 또는 이미 지원한 업체의 공고 숨기기
- 사용자가 수집 버튼을 눌렀을 때 지원 현황 페이지에서 지원 이력 읽기
- 공고에서 우클릭으로 제외 업체를 등록하는 메뉴 제공

확장 프로그램은 위 기능에 필요한 페이지 내용만 처리합니다.

## 사용자 제어

사용자는 JobFilter 관리 화면에서 저장된 데이터를 관리할 수 있습니다.

관리 가능한 항목은 다음과 같습니다.

- 제외 업체 해제
- 저장된 지원 이력 삭제
- 선택한 날짜의 지원 이력 일괄 삭제
- 지원 이력 CSV 내려받기
- 지원 이력 CSV 등록
- popup에서 제외 업체 숨김 및 지원완료 업체 숨김 설정 변경

확장 프로그램을 삭제하거나 Chrome에서 확장 프로그램의 저장 데이터를 삭제하면 로컬에 저장된 데이터도 제거할 수 있습니다.

## 권한 설명

JobFilter는 다음 Chrome 권한을 사용합니다.

- `storage`: 제외 업체, 지원 이력, 설정을 브라우저 로컬 저장소에 저장하기 위해 사용합니다.
- `contextMenus`: 공고에서 우클릭으로 업체를 제외하는 메뉴를 제공하기 위해 사용합니다.
- `activeTab`: 현재 탭에서 지원 이력을 수집하거나 필터 상태를 갱신하기 위해 사용합니다.

host permissions는 지원 사이트에서 공고 필터링과 지원 이력 수집을 수행하기 위해 사용합니다.
